# **App Name**: DeebData Trading

## Core Features:

- Landing Page: A sleek, professional homepage featuring a hero section, feature cards, and a live Ticker Tape from TradingView integrated at the top for real-time market data display.
- AI Signal Dashboard: A private user dashboard displaying AI-generated Buy/Sell signals for Forex (Gold, EUR/USD) and Crypto (BTC, ETH), including Entry, Take Profit (TP), Stop Loss (SL) values, and a 'Confidence Score' tool generated by AI.
- TradingView Widgets Integration: Dynamic integration of TradingView's Economic Calendar and Market Summary widgets to provide users with comprehensive market insights.
- Free Trial System: A system allowing new users to access and view exactly 7 AI-generated signals for free before requiring a subscription.
- Subscription Paywall: A 'Glassmorphism' style popup that appears after the free trial limit, prompting users to 'Unlock Unlimited VIP Signals for $25/Month' to continue access.
- WhatsApp Support: A floating WhatsApp button visible across the application, providing users with instant access to customer support.
- Hidden Admin Panel: A secret admin interface, accessible via 7 clicks on the main logo, secured by a password, allowing administrators to post new signals and view total user statistics.

## Style Guidelines:

- Primary background color: Pure Black (#000000) for a deep, high-contrast trading dashboard aesthetic.
- Secondary background and neutral elements: Slate Gray, using a hex value like #2C2E33 for contrast against black.
- Primary interactive color: A vibrant yet professional blue (#99B2FA) for calls to action, active states, and key branding elements, providing strong contrast on dark backgrounds.
- Accent and highlight color: A bright, clear cyan (#33BCD8) for secondary emphasis and high-tech flair.
- Semantic success indicators: Neon Green (#00FF00) for positive signals and achievements.
- Semantic alert indicators: Neon Red (#FF0000) for warnings, errors, and negative signals.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly legible text experience suitable for data-intensive dashboards.
- Utilize a consistent set of crisp, professional outline or filled icons that complement the dark theme and high-tech aesthetic.
- Responsive design adapting seamlessly for both mobile and desktop screens, featuring a fixed header, clear navigation, and well-organized data display areas.
- Subtle and fluid animations for transitions, hover effects, and the 'Glassmorphism' paywall popup, enhancing the high-tech and premium user experience.
